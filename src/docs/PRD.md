# RecallRadar - Product Requirements Document

## Vision

RecallRadar is an intelligent learning retention system designed to help learners identify and address knowledge decay before it negatively impacts performance.

The system focuses on retention, revision effectiveness, confidence accuracy, and learning consistency rather than simple note-taking or task management.

---

## Problem Statement

Most learners forget information gradually after studying. This forgetting often becomes visible only during interviews, exams, coding assessments, or real-world application.

RecallRadar aims to detect learning decay before failure occurs and provide actionable revision recommendations.

---

## Target Users

- Students
- Software Engineers
- Competitive Exam Aspirants
- Lifelong Learners

---

## MVP Scope

### Topic Management

- Create Topic
- View Topic
- Update Topic
- Delete Topic

### Revision Tracking

- Log Revision Sessions
- Track Revision Method
- Track Confidence Level

### Retention Analytics

- Leak Score Calculation
- Revision Priority Ranking
- Weakest Topics Identification

---

## Out Of Scope (MVP)

- Mobile Application
- AI Predictions
- Social Features
- Team Collaboration
- Advanced Machine Learning Models

---

## Success Criteria

A user should be able to:

1. Create learning topics.
2. Record revision sessions.
3. View topics requiring urgent revision.
4. Track retention trends over time.
