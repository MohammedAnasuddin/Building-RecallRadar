# RecallRadar

> Intelligent Learning Retention & Revision Tracking System

RecallRadar is a backend-focused Spring Boot application designed to help learners identify what they are forgetting before it becomes a problem.

Instead of only tracking what users learn, RecallRadar focuses on:

- retention
- memory decay
- revision effectiveness
- learning consistency
- revision prioritization

The long-term goal is to provide actionable insights into a learner's knowledge retention using concepts inspired by cognitive science and learning research.

---

## Problem

Most learners:

- study once
- feel confident temporarily
- gradually forget information over time

The challenge is that forgetting often becomes visible only during:

- interviews
- exams
- coding assessments
- real-world application

RecallRadar aims to detect learning decay before failure occurs.

---

## Core Concepts

The project is inspired by:

- Spaced Repetition
- Active Recall
- Forgetting Curve
- Retrieval Strength Theory
- Learning Analytics

Future versions will estimate:

- memory leak score
- revision urgency
- confidence accuracy
- retention trends
